Química: Reações

Tema: Química Dificuldade: básico–intermediário Tempo estimado: 60–90 min

Identificação de tipos de reação, balanceamento por inspeção e aplicações cotidianas.

Objetivos

  • Classificar reações comuns (síntese, decomposição, simples/dupla troca, combustão, neutralização).
  • Balancear equações simples pelo método de inspeção.
  • Aplicar conservação da massa e de átomos.

Teoria

Tipos de reações

  • Síntese: A + B → AB
  • Decomposição: AB → A + B
  • Simples troca: A + BC → AC + B
  • Dupla troca: AB + CD → AD + CB
  • Neutralização: ácido + base → sal + água
  • Combustão (de hidrocarbonetos): CxHy + O₂ → CO₂ + H₂O

Balanceamento

  • Conservar número de átomos de cada elemento em ambos os lados (coeficientes inteiros mínimos).
  • Método de inspeção: ajustar por tentativa organizada (comece pelos elementos que aparecem menos).

Conservação

  • Lei da conservação da massa (Lavoisier): a massa se conserva em sistemas fechados.

Erros comuns

  • Alterar índices (fórmulas) em vez de usar coeficientes.
  • Esquecer de balancear O e H ao final (em combustão/neutralização).

Exemplos resolvidos

Exemplo 1 — Combustão do etano (C₂H₆)
C₂H₆ + O₂ → CO₂ + H₂O
Balanceando: C:2 → 2CO₂; H:6 → 3H₂O; O: 2×2 + 3×1 = 7 O → 7/2 O₂; multiplicando por 2: 2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O.
Exemplo 2 — Neutralização
H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O.

Exercícios

1) Classifique: CaCO₃ → CaO + CO₂.
GabaritoDecomposição.
2) Balanceie: Fe + O₂ → Fe₂O₃.
Gabarito4Fe + 3O₂ → 2Fe₂O₃.
3) Identifique o tipo: Zn + CuSO₄ → ZnSO₄ + Cu.
GabaritoSimples troca.
4) Balanceie: Al₂(SO₄)₃ + Ca(OH)₂ → Al(OH)₃ + CaSO₄.
GabaritoAl₂(SO₄)₃ + 3Ca(OH)₂ → 2Al(OH)₃ + 3CaSO₄.
5) Reação: HCl + NaHCO₃ → ? (produtos principais).
GabaritoNaCl + H₂O + CO₂ (dupla troca + decomposição do H₂CO₃).
